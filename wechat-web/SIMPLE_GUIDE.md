# 📚 公众号文章管理工具 - 使用指南

## 🎯 核心功能

这是一个简洁实用的公众号文章管理工具，专注于三个核心功能：

```
1️⃣ 订阅公众号 → 自动拉取文章
2️⃣ 收藏好文章 → 永久保存
3️⃣ 快速搜索 → 秒找文章
```

---

## 🚀 快速开始

### 步骤1：订阅公众号

1. 打开工具
2. 左侧边栏点击 "➕ 添加公众号"
3. 输入公众号名称
4. 点击搜索并添加

**效果**：
- 公众号添加到订阅列表
- 自动拉取最新文章

---

### 步骤2：浏览和收藏

1. 点击左侧公众号名称
2. 浏览文章列表
3. 点击文章卡片阅读
4. 觉得好文 → 点击 **⭐ 收藏**

**收藏位置**：
- 阅读页面底部 "⭐ 收藏" 按钮
- 或文章列表的 ⭐ 图标

---

### 步骤3：搜索收藏

1. 点击左侧 "⭐ 我的收藏"
2. 使用顶部搜索框
3. 输入关键词（标题、作者、内容）
4. 实时显示结果

**搜索范围**：
- ✅ 文章标题
- ✅ 作者名称
- ✅ 文章摘要

---

## 📋 功能详解

### 1️⃣ 订阅管理

#### 添加公众号
```
左侧边栏 → ➕ 添加公众号 → 搜索 → 添加
```

#### 查看文章
```
点击公众号名称 → 显示文章列表
```

#### 删除订阅
```
右键公众号 → 删除（或长按）
```

---

### 2️⃣ 收藏功能

#### 收藏文章
**方法1**：文章列表
- 点击文章卡片的 ⭐ 图标

**方法2**：阅读页面
- 点击底部 "⭐ 收藏" 按钮

#### 取消收藏
- 再次点击 ⭐ 按钮

#### 查看收藏
```
左侧 "⭐ 我的收藏" → 显示所有收藏
```

**收藏数据**：
- 保存在浏览器 LocalStorage
- 刷新页面不丢失
- 清除缓存会清空

---

### 3️⃣ 搜索功能

#### 基础搜索
```
我的收藏 → 顶部搜索框 → 输入关键词 → 实时显示
```

**搜索内容**：
- 标题关键词
- 作者名称
- 文章摘要

**特点**：
- ⚡ 实时搜索
- 🔤 不区分大小写
- 📝 支持中文和英文

#### 筛选功能

**按状态筛选**：
- 全部
- 未读
- 已读

**按标签筛选**：
- 自动提取标签
- 点击标签快速筛选

## ❓ 常见问题

### Q: 收藏的文章会丢失吗？
**A**: 不会，除非清除浏览器缓存。建议定期导出备份。

### Q: 可以在多台设备同步吗？
**A**: 需要手动导出/导入JSON文件。暂不支持云同步。

### Q: 搜索支持模糊匹配吗？
**A**: 是的，输入部分关键词即可搜索。

### Q: 如何删除收藏？
**A**: 再次点击 ⭐ 按钮即可取消收藏。

### Q: 相关推荐是如何工作的？
**A**: 从当前账号的文章中随机推荐3篇。
