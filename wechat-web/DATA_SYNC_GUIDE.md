# 📊 数据同步指南

## 问题：换浏览器数据就没了？

**原因**：数据存储在浏览器的 LocalStorage，不同浏览器之间数据独立。

**解决方案**：使用导出/导入功能！

---

## 🚀 快速开始

### 方法1：在设置页面操作

1. 点击左侧 **"⚙️ 设置"**
2. 在 **"数据管理"** 区域：
   - 点击 **"📤 导出数据"** → 下载备份文件
   - 点击 **"📥 导入数据"** → 选择备份文件

### 方法2：使用快捷键（更快！）

```
Cmd/Ctrl + E  →  导出数据
Cmd/Ctrl + I  →  导入数据
```

---

## 📝 使用场景

### 场景1：换浏览器使用

**步骤**：
1. **旧浏览器**：Cmd/Ctrl + E 导出数据
2. **新浏览器**：打开工具 → Cmd/Ctrl + I 导入数据
3. **完成**：刷新页面，数据全部恢复

### 场景2：换电脑使用

**步骤**：
1. **旧电脑**：导出数据 → 保存到云盘/U盘
2. **新电脑**：下载备份文件 → 导入数据
3. **完成**：所有数据同步完成

### 场景3：定期备份

**建议**：
- 每周导出一次数据
- 保存到云盘（如百度云、OneDrive）
- 避免数据丢失

---

## 📦 导出的数据包含

导出的 JSON 文件包含：

```json
{
  "version": "1.0",
  "exportDate": "2025-11-04T12:00:00Z",
  "favorites": [],      // 收藏的文章
  "accounts": [],       // 订阅的公众号
  "settings": {},       // 个人设置

```

## ⚠️ 注意事项

### 1. 导入会覆盖当前数据

**说明**：导入数据会覆盖当前浏览器的所有数据

**建议**：导入前先导出当前数据备份

### 2. 浏览器兼容性

**支持的浏览器**：
- ✅ Chrome/Edge
- ✅ Firefox
- ✅ Safari
- ✅ Opera

### 3. 数据安全

**说明**：
- 数据仅保存在本地，不上传服务器
- 备份文件请妥善保管
- 不要分享给不信任的人

## 🔧 故障排查

### Q1: 导出失败

**可能原因**：
- 浏览器不支持文件下载
- 权限不足

**解决方法**：
- 检查浏览器设置
- 尝试换个浏览器

### Q2: 导入失败

**可能原因**：
- JSON 格式错误
- 文件损坏

**解决方法**：
- 用文本编辑器检查JSON格式
- 重新导出数据

### Q3: 导入后数据没变化

**可能原因**：
- 没有刷新页面

**解决方法**：
- Cmd/Ctrl + Shift + R 强制刷新

## 💬 常见问题

### Q: 数据会上传到服务器吗？
**A**: 不会！所有数据只保存在你的浏览器本地。

### Q: 可以恢复到某个时间点吗？
**A**: 可以！只要你保存了那个时间点的备份文件。

### Q: 导入会删除新数据吗？
**A**: 会覆盖！建议导入前先导出当前数据。

### Q: 可以选择性导入吗？
**A**: 暂不支持，计划未来版本添加。
